# דף נחיתה – חישוב עלויות ומחירים (עברית, מינימלי)

אפליקציה קטנה לשימוש חוזר:
- מעלים קובץ אקסל (xlsx)
- עורכים מספרים (חומרים/שעות/תוספות/כמות)
- רואים תוצאות בזמן אמת
- מייצאים ל־PDF או תמונת PNG
- מורידים גם אקסל מעודכן

## איך מריצים (מחשב מקומי)
1) התקנת פייתון 3.10+  
2) התקנת תלויות:
```bash
pip install -r requirements.txt
```
3) הרצה:
```bash
streamlit run app.py
```

## הערות
- החישוב נעשה בתוך האפליקציה כדי לעבוד גם בלי Excel מותקן.
- אם תרצה “תבנית עיצוב” יותר ספציפית (לוגו/טיפוגרפיה/מידות), אפשר לשדרג את ה־PDF/PNG.
